# 4. 디버깅 보고서 (10점)

> 💡 **디버깅이란?** 프로그램에서 발생한 오류(버그)를 찾아내고 고치는 과정입니다.

---

## 📝 작성 정보
- **작성 날짜:** 2025년 _11_월 _20_일
- **작성자:** (고동훈)

---


## 🧪 프로그램 테스트 결과

### 테스트 1
- **무엇을 테스트했나요?** 이름을 적으면 등록이 되는것을 테스트 함.
- **어떻게 테스트했나요?** 이름을 입력하면 프로그램이 이름을 등록하여, 없는 이름은 작동되지 않게 테스트하였다.
- **결과:** ✅ 성공 

### 테스트 2
- 등록된 선수 이름을 적었을 때 선수의 정보와 성적을 입력 할수있게 칸을 뜨게 하였다.
- 이름 칸에 선수 이름을 넣고 작동 했을 때 선수의 정보와 성적을 나오게 하였다.
- **결과:** ✅ 성공

## 💭 디버깅하면서 배운 점

디버깅으로 내가 원하는 프로그램을 만들수 있다는것에 대해 새롭게 알게되었다.
