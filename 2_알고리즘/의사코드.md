# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
시작

선수 정보를 저장할 구조체 배열 players[100] 선언
현재 선수 수 count = 0

반복문 시작
    메뉴 출력:
        1. 선수 정보 입력
        2. 선수 목록 출력
        3. 선수 검색
        4. 통계 보기 (최고 득점자 / 평균 득점)
        0. 종료
    사용자로부터 메뉴 입력 → menu

    만약 menu == 1이면
        addPlayer(players, count) 호출
    아니면 menu == 2이면
        listPlayers(players, count) 호출
    아니면 menu == 3이면
        searchPlayer(players, count) 호출
    아니면 menu == 4이면
        showStats(players, count) 호출
    아니면 menu == 0이면
        "프로그램 종료" 출력
        반복문 종료
    그렇지 않으면
        "잘못된 입력입니다." 출력
반복문 종료

끝


## 2. 핵심 기능 1: 선수 정보 입력
사용자에게 선수 이름 입력
사용자에게 포지션 입력
사용자에게 득점 수 입력
사용자에게 출전 경기 수 입력

입력받은 정보를 players[count]에 저장
count = count + 1

"선수 정보가 추가되었습니다." 출력


## 3. 핵심 기능 2: 선수 목록 출력
만약 count == 0이면
    "등록된 선수가 없습니다." 출력
그렇지 않으면
    0부터 count-1까지 반복하면서
        players[i]의 이름, 포지션, 득점, 경기 수 출력
